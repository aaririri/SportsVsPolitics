# Sports vs. Politics Text Classifier
An NLP project comparing ML techniques for document classification.

## Project Overview
This project classifies news articles into two categories using TF-IDF and N-Grams.

## Models Compared
1. **Multinomial Naive Bayes**
2. **Support Vector Machine (SVM)**
3. **Random Forest**
